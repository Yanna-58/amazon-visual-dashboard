## 📤 管理员分发流程 (v3.2)

```
1️⃣ 打开 v3.2 最新版 → 加载数据 → 分配设计师标签
2️⃣ 点击顶部【📤 导出配置】→ 得到 JSON 文件
3️⃣ 将 JSON 文件 + 在线链接 发送给同事
4️⃣ 同事打开在线链接 → 上传数据 → 点【📥 导入配置】
5️⃣ 所有人标签设置一致 ✅
```
