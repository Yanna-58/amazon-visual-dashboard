## ✨ 这是什么？

**亚马逊视觉数据看板**是一个纯前端的 HTML 数据分析工具，**无需安装、无需注册、无需服务器**。下载即用。

它解决了一个痛点：**设计师做完图之后，不知道效果好不好，只能凭感觉。** 有了这个工具，上传亚马逊后台数据 → 一眼看出哪个产品的 CTR/CVR 偏低 → 判断是不是主图/A+的问题 → 优化后对比验证。

## 🎯 核心功能

| 功能 | 说明 |
|------|------|
| 📊 **产品总览** | 所有 ASIN 一目了然，带健康度色标 🟢🟡🔴 |
| 📈 **趋势变化** | 每个产品显示访/转/销环比涨跌 ↑+21% / ↓-15% |
| 👤 **设计师标签** | 给 ASIN 分配设计师，按人筛选管理 |
| 🔍 **新品追踪** | CTR/CVR 趋势 + 视觉诊断 + 品类基准对比 |
| 📅 **环比分析** | CTR/CVR/流量/广告花费/CPC 环比 + 视觉解读 |
| 📆 **同比分析** | 今年同月 vs 去年同月 + 百分比变化对比 |
| ⚖️ **优化对比** | 优化前后数据对比（BR+Ads 双文件）+ 视觉归因 |
| 🌍 **站点切换** | 美/欧/日站货币自动切换 + 服装类目基准 |
| 💾 **数据持久化** | localStorage 自动保存，刷新不丢数据 |
| 🖨️ **导出** | 打印 PDF + CSV 导出 |
| 🌙 **暗色模式** | 护眼模式 |
| 📤 **导出配置** | 导出设计师标签/ASIN分配为 JSON 文件（管理员分发用） |
| 📥 **导入配置** | 一键导入管理员分发的配置文件 |

## 🚀 快速上手

### 方式一：在线使用（推荐）

👉 **[点击这里直接打开看板](https://yanna-58.github.io/amazon-visual-dashboard/amazon-visual-dashboard.html)**

无需下载，浏览器打开即用。数据仅保存在本地浏览器中。

### 方式二：本地下载

1. 下载 [amazon-visual-dashboard.html](amazon-visual-dashboard.html)
2. 双击用浏览器打开
3. 点击「加载新品演示数据」快速体验
4. 或上传你的亚马逊业务报告 CSV

### 方法二：从亚马逊下载数据

```
亚马逊卖家后台 → 业务报告 → 详情页面销售和流量
→ 选择「按父ASIN」或「按子ASIN」→ 下载CSV
→ 直接上传到看板（不需要修改任何格式！）
```

## 📋 支持的数据格式

看板使用**模糊匹配**，以下表头都能识别：

| 必要字段 | 可接受的表头（举例） |
|---------|-------------------|
| 日期 | Date / 日期 / 时间 |
| 会话数 | Sessions / Session Count / 会话数 |
| 转化率 | Unit Session Percentage / CVR / 转化率 |
| 销售额 | Ordered Product Sales / Sales / 销售额 |
| ASIN | ASIN / Parent ASIN / 父ASIN |

> 详细模板说明请查看 [导入模板说明.md](导入模板说明.md)

## 🏗️ 技术栈

- **纯前端**：HTML5 + CSS3 + Vanilla JavaScript
- **零依赖**：无框架、无 npm、无构建步骤
- **图表库**：Chart.js (CDN)
- **数据存储**：localStorage
- **文件大小**：~200KB 单个 HTML 文件
