# 用户操作手册

## 亚马逊商品视觉数据看板软件 V3.2

> 本手册为「亚马逊商品视觉数据看板」的用户操作指引，帮助用户快速上手并充分利用各项功能。

---

## 一、快速开始（3分钟上手）

### 第1步：打开软件

双击 `亚马逊视觉数据看板_v3.2(最新版).html` 文件，或用浏览器（推荐 Chrome/Edge）打开该文件。

> 💡 **提示**：本软件是纯前端应用，无需联网即可使用核心功能。联网仅用于加载 Chart.js 图表库。

### 第2步：导入数据

1. 点击顶部导航栏的 **「📤 数据导入」** 标签页
2. 点击 **「点击上传 Business Report CSV」** 按钮
3. 选择从亚马逊后台下载的 `.csv` 文件
4. （可选）同样方式上传广告报表
5. 上传成功后，系统会自动跳转到分析页面

### 第3步：查看分析结果

- **新品追踪** — 查看整体 KPI 趋势和视觉诊断
- **产品总览** — 如果上传的是"按ASIN"报告，可在此查看每个产品的表现
- **环比分析** — 查看近期指标变化趋势
- **优化对比** — 上传优化前后的数据，对比效果

## 二、数据准备指南

### 2.1 如何从亚马逊后台下载 Business Report

```
卖家后台（Seller Central）
    └── 数据报告（Business Reports）
        └── 业务报告（Business Reports）
            ├── 选择报告类型：「详情页面销售和流量」（Detail Page Sales and Traffic）
            ├── 选择维度：
            │   ├── 「按日期」（By Date）→ 整店趋势分析
            │   └── 「按ASIN」（By ASIN）→ 产品级别分析 ⭐ 推荐
            ├── 设置时间范围
            └── 点击「下载」（Download）→ 获得 .csv 文件
```

### 2.2 如何下载广告报表

```
卖家后台（Seller Central）
    └── 广告活动管理（Campaign Manager）
        └── 衡量（Measure）
            └── 创建报告（Create Report）
                ├── 选择报告类型：推广商品报告 / 搜索词报告
                ├── 设置时间范围
                └── 下载 CSV
```

### 2.3 必要字段检查清单

| 报告类型 | 必须有的列 | 可选但建议有的列 |
|----------|-----------|----------------|
| Business Report | ✅ Date（日期）<br>✅ Sessions（会话数）<br>✅ Unit Session %（转化率） | Page Views<br>Buy Box %<br>Units Ordered<br>Ordered Product Sales |
| 广告报表 | ✅ Impressions（曝光量）<br>✅ Clicks（点击数）<br>✅ Spend（花费） | CTR、CPC<br>ACOS、Sales |

### 2.4 常见问题排查

| 问题 | 原因 | 解决方法 |
|------|------|---------|
| 提示"缺少日期列" | 下载的不是业务报告 | 确认在「业务报告」页面下载 |
| 中文乱码 | 编码不匹配 | 软件已支持自动检测编码，如仍有问题请尝试另存为 UTF-8 格式 |
| 数据行数为0 | 可能只下载了表头 | 确保选择了有数据的日期范围 |
| ASIN未识别 | 表头中不含ASIN关键词 | 使用"按ASIN"维度的报告格式 |

## 三、功能模块详细操作

### 3.1 📊 新品追踪（Dashboard）

#### KPI 卡片解读

| 指标 | 视觉含义 | 正常范围参考（服装类目美国站） |
|------|---------|----------------------------|
| **Sessions（会话数）** | 多少人看到了你的产品 | — |
| **CVR（转化率）** | 看到的人有多少下单了 | 8% - 15% 为健康水平 |
| **Page Views（浏览量）** | 总共被浏览了多少次 | PV/Session ≥ 2.5 较好 |
| **Sales（销售额）** | 总销售金额 | — |

卡片下方的小标签显示**环比变化**：
- 🟢 绿色 + 数字 = 上升（好趋势）
- 🔴 红色 - 数字 = 下降（需关注）
- 灰色 → = 持平

#### 视觉健康诊断

系统根据你的 CVR 和 PV/Session 自动给出诊断：

| 诊断等级 | 条件 | 建议 |
|---------|------|------|
| ✅ **优秀** | CVR ≥ 12% 且 PV/Session ≥ 2.5 | 保持当前设计方向 |
| ⚠️ **中等** | CVR 6%-12% 或 PV/Session 接近阈值 | 建议增加场景图/A+内容 |
| 🔴 **偏低** | CVR < 6% 或两项均低 | 需重点优化主图和详情页 |

#### CTR-CVR 四象限矩阵

这是帮你快速判断**优先优化什么**的工具：

| | 低 CTR（主图不够吸引）| 高 CTR（主图OK）|
|--|---|---|
| **高 CVR（详情页OK）**| 🟡 主图待优化 — 重点改主图构图/颜色 | 🟢 **优秀** — 保持即可 |
| **低 CVR（详情页需改进）**| 🔴 **全面优化** — 主图+详情都要改 | 🟡 详情差 — 重点优化副图和A+ |

#### 品类基准对比

底部显示你的指标 vs 行业基准值：
- 🟢 **绿色** = 你优于基准
- 🔴 **红色** = 你低于基准

### 3.2 🏷️ 产品总览（Products）

此页面仅在**上传了"按ASIN"格式的报告**时才会显示数据。

**操作步骤：**

1. 上传按ASIN维度的 Business Report
2. 自动进入「产品总览」页
3. 查看每个产品的：
   - 健康度指示灯（🟢🟡🔴）
   - ASIN 编号和产品名称
   - 核心KPI数值
   - 近期趋势标签
4. **点击表格中的任意行** → 跳转到该产品的详细追踪页
5. **点击👤图标** → 给该产品分配设计师姓名

**排序功能：** 点击表头可按任意列排序（默认按会话数降序）

**设计师筛选：**
- 下拉框选择特定设计师 → 只显示其负责的产品
- 彩色标签快速筛选

### 3.3 📈 环比 & 同比分析

#### 环比模式（本月 vs 上月）

适合追踪**短期变化趋势**，发现近期的异常波动。

**五大指标解读：**

| 指标 | 上升意味着 | 下降意味着 |
|------|-----------|-----------|
| **CTR（点击率）** | 主图更吸引了 | 可能竞品换图了，或季节因素 |
| **CVR（转化率）** | 详情页说服力增强 | 详情页可能过时了 |
| **流量（会话数）** | 曝光效果提升 | 整体流量在下降 |
| **广告花费** | 投入增加 | 投入减少 |
| **CPC（单次点击成本）** | 竞争加剧 | 竞争缓和 |

#### 同比模式（今年 vs 去年同月）

⭐ **老品必看同比！**

原因：环比受季节性影响大（比如服装行业3月是淡季、4月是旺季），同比才能排除季节因素看到**真实的年度竞争力变化**。

**操作方法：**
1. 点击「📅 同比」按钮
2. 分别上传去年同月和今年同月的 Business Report
3. 系统自动计算同比差异

### 3.4 ⚡ 优化对比

**用途：** 当你对某个产品做了视觉优化（换主图、改A+等）后，用此功能量化验证优化效果。

**操作流程：**
1. 在优化前 → 导出当时的 Business Report 和广告报表
2. 完成视觉优化并上线
3. 等 2-4 周（让新图片积累足够的数据）
4. 导出相同时段的报告
5. 分别上传到「优化前」和「优化后」区域
6. 查看对比结果

**结果判读：**

| CVR 变化 | 判定 | 建议动作 |
|---------|------|---------|
| 提升 > 3个百分点 | 🎉 效果显著 | 记录成功经验，复用到其他产品 |
| 提升 0-3 个百分点 | 📊 有一定效果 | 继续观察，考虑微调 |
| 无提升或下降 | ⚠️ 效果不明显 | 回顾改动内容，尝试不同方向 |

## 四、工具栏功能说明

### 打印报告
点击后将隐藏所有操作按钮，生成适合打印/PDF导出的干净报告页面。按 `Ctrl+P` 即可保存为 PDF。

### 导出 CSV
将当前分析的所有数据（含产品总览、原始数据）导出为 CSV 文件，可用 Excel 打开进一步编辑。

### 导出配置
将你设置的**设计师分配信息**和**ASIN产品列表**导出为 JSON 文件。方便：
- 在多台电脑间同步配置
- 与团队同事分享产品-设计师对应关系

### 导入配置
加载之前导出的 JSON 配置文件，恢复设计师标签等设置。

### 重置
清除所有本地缓存的数据，恢复到初始状态。⚠️ 此操作不可撤销！

### 暗色模式
切换亮色/暗色主题。暗色模式适合长时间使用或在光线较暗的环境下使用。

### 站点切换
在美国站🇺🇸/欧洲站🇬🇧/日本站🇯🇵之间切换。切换后会更新货币符号（$/£/¥）和品类基准值。

### 时间范围
选择分析的时间窗口：7天 / 14天 / 30天 / 90天。时间范围越长，趋势越稳定；越短，对最新变化越敏感。

## 六、常见问题 FAQ

**Q: 数据会上传到服务器吗？**
A: 不会。所有数据处理都在你的浏览器本地完成，没有任何数据离开你的计算机。

**Q: 支持哪些浏览器？**
A: 推荐 Chrome、Edge、Firefox、Safari 的较新版本。不支持 IE 浏览器。

**Q: 可以同时上传多少个文件？**
A: 不限制。可以多次上传不同的数据集，每次新上传会覆盖旧的同类型数据。

**Q: 关闭浏览器后数据还在吗？**
A: 在！数据保存在浏览器的 localStorage 中。只要不清除浏览器数据，下次打开看板时数据会自动恢复。（除非点了「重置」按钮）

**Q: 为什么有些指标显示"—"？**
A: 表示还没有上传对应类型的数据。例如广告指标需要上传广告报表后才会显示。

**Q: 支持领星ERP导出的数据吗？**
A: 只要导出格式是标准CSV且包含所需关键字段（Sessions/CVR/Date等），就能识别。如果字段名不完全一致，软件的模糊匹配功能也能处理大部分情况。
