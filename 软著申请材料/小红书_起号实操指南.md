## 六、新手最容易犯的错误 ❌

| 错误 | 正确做法 |
|------|---------|
| 第一篇就追求完美 | 先发出去，完成比完美重要 |
| 发完不停刷新看数据 | 发完就去干别的，别刷！ |
| 看到阅读量低就放弃 | 新号前10篇都是"交学费"期 |
| 一次发太多标签 | 5-10个精准标签就够了 |
| 只发不互动 | 评论一定要回！互动率影响推荐 |
| 封面随便截个图 | 封面决定80%点击率，认真做！ |
| 内容太"广告感" | 要像朋友聊天一样分享，别像卖货 |
