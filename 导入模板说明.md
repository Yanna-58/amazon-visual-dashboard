# 📋 亚马逊视觉数据看板 — 数据导入模板说明

> 不用担心格式问题！看板用的是**模糊匹配**，只要表头里包含关键词就能识别，不需要一字不差。

---

## 🔑 核心原则

| 原则 | 说明 |
|------|------|
| **模糊匹配** | 表头只要**包含**关键词即可，比如 `Session` 能匹配 `Sessions`、`Session Percentage` 等 |
| **中英双语** | 中文表头也能识别！比如 `日期`、`会话`、`转化率` 都可以 |
| **多余列没关系** | 你导出的表格有多余的列完全不影响，看板只读需要的字段 |
| **格式支持** | CSV 和 TSV 都行，从亚马逊后台直接下载的 .txt 文件也支持 |
| **编码自动检测** | UTF-8、GBK 等常见编码都能正确识别 |

## ❓ 常见问题

### Q1：我从亚马逊下载的文件是 .txt 格式，能上传吗？
**能！** 亚马逊下载的 .txt 文件本质上是 TSV（Tab分隔），看板完全支持。

### Q2：表头名称不一样怎么办？
**不用担心！** 看板用模糊匹配，只要表头**包含**关键词就行。比如：
- ✅ `Sessions` → 包含 `session` → 识别
- ✅ `Session Count` → 包含 `session` → 识别  
- ✅ `会话数` → 包含 `会话` → 识别
- ✅ `Unit Session Percentage` → 包含 `unit session` → 识别

### Q3：上传后提示缺少字段怎么办？
看板会明确告诉你缺了哪个字段，比如：
> ⚠️ 缺少"日期(Date)"列 — 趋势图和环比分析将无法工作

你只需要回到亚马逊后台确认导出时选对了报告类型就行。

### Q4：数据格式需要特别处理吗？
**不需要！** 直接从亚马逊下载，不要修改任何内容，直接上传即可。
- 日期格式：`2026-04-01` 或 `04/01/2026` 都行
- 百分比格式：`12.5%` 或 `0.125` 都行
- 金额格式：`2589.36` 或 `$2,589.36` 都行

### Q5：数据量有限制吗？
建议单次上传不超过 2000 行。如果你的数据量特别大，可以缩短日期范围分批分析。

```
1. 亚马逊后台 → 数据报告 → 业务报告 → 下载

3. （可选）再上传广告数据

```
